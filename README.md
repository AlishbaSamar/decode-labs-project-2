# DecodeLabs Backend API - Project 2

**Batch 2026 | Full Stack Development Internship**

A robust Express.js backend API for the DecodeLabs internship application system. This API handles user registration, validation, and management for internship applications.

## Features
- RESTful API endpoints for CRUD operations
- Comprehensive input validation (name, email, message)
- CORS enabled for cross-origin requests
- Professional error handling
- In-memory data storage
- JSON request/response format

## Tech Stack
- **Runtime**: Node.js
- **Framework**: Express.js (v5.2.1)
- **Middleware**: CORS
- **Port**: 5000

## Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)
